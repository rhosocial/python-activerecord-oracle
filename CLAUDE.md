# Project Overview: rhosocial-activerecord-oracle

## Project Name
- **Repository Name**: python-activerecord-oracle
- **Python Package Name**: rhosocial-activerecord-oracle

## WARNING

**This project is currently under active development and is NOT ready for production use.**

- **DO NOT use in testing environments**
- **DO NOT use in production environments**
- **DO NOT rely on any functionality as it may change without notice**

## Project Purpose

This project is an Oracle backend implementation for the `rhosocial-activerecord` Python package. It provides Oracle database support with the elegant ActiveRecord pattern interface.

## Key Design Principles

1. **Backend Implementation**: Extends core ActiveRecord with Oracle-specific features
2. **Driver**: Uses `oracledb` for database connectivity
3. **Namespace Package**: Integrates with the rhosocial namespace package architecture

## Current Status

This project is in early development. Key features being implemented:

- Basic CRUD operations
- Connection management
- Transaction support
- Schema introspection
- Oracle-specific data types

## Version Control and Changelog

This project adheres to the same version control, branching, commit message, and changelog management standards as the main `python-activerecord` project.
